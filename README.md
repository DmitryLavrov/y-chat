## Getting Started

### Prerequisites

```shell
nvm install lts
nvm use lts
npm i npm@latest -g
npm init -y
```

### Installation

```shell
$ npm i -D parcel
$ npm i -D handlebars
$ npm i -D @anikin/parcel-transformer-handlebars
$ npm i -D @anikin/parcel-transformer-handlebars-precompile
$ npm i -D pug
$ npm i -D sass

$ npm i -D express
```
